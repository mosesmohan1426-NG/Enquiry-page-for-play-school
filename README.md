# Little Roses Play School

A modern, fully responsive website built for **Little Roses Play School** — 
designed to give parents a warm first impression and easy access to admissions info.

##Features

- **Animated Preloader** — smooth branded intro on page load
- **Particle Canvas Background** — floating particle animation on the hero
- **Responsive Navbar** — hamburger menu for mobile with smooth scroll
- **Counter Animation** — animated stats (Happy Kids, Years of Care, Staff)
- **Gallery with Lightbox** — filterable photo gallery with lightbox viewer
- **Testimonials Slider** — auto-scrolling parent reviews
- **Contact Form → Google Sheets** — form submissions saved via Google Apps Script
- **Scroll Reveal Animations** — sections animate in as you scroll

## Tech Stack

| Layer | Tech |
|-------|------|
| Markup | HTML5 |
| Styling | CSS3 (custom, no framework) |
| Logic | Vanilla JavaScript |
| Backend | Google Apps Script (form data → Google Sheets) |
| Hosting | Netlify |

## Setup

1. Clone the repo
2. Open `js/main.js` and replace the `SHEETS_URL` with your own Google Apps Script Web App URL
3. Deploy on Netlify or open `index.html` directly in a browser
